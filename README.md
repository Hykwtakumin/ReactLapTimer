# ReactLapTimer

LapTimerをReact + TypeScriptで実装

this project uses:
 - typescript (ES6以上の新記法への対応/静的型付け)
 - react (VDOMを用いたViewの描画)
 - create-react-app (お手軽にreactアプリを作れるボイラープレート)
 - CodeSandBox (Cloud IDE)
 
 this project does NOT use:
 - yeomen
 - Webpack
 - gulp
